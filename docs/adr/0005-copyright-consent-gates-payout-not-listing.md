# Copyright consent gates Payout, not listing

A design that uses copyrighted content can still be listed, browsed, and ordered normally — customers are charged, Print Batches close, and items ship — even before the designer has obtained consent from the copyright holder. Consent only gates whether that design's earnings are ever disbursed via Payout; unconsented earnings sit in the designer's Balance indefinitely.

We chose this over blocking the design from being orderable at all, because gating every listing behind a manual copyright review would undermine the "post and get promoted by default" flow for the common case of original designs. The accepted risk: if consent is never obtained, the platform will have shipped real physical products built on potentially infringing content, while only withholding payment from the designer — not preventing the infringement itself. We're accepting that risk deliberately rather than slowing down every designer to police a minority case.
