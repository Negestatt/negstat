# Triage Label Vocabulary

The following default labels are used by the `triage` skill to manage the state of issues:

*   **`needs-triage`**: The issue requires evaluation by a maintainer.
*   **`needs-info`**: The issue is awaiting further information from the reporter.
*   **`ready-for-agent`**: The issue is fully specified and ready for an AI agent to pick up and work on without requiring human context.
*   **`ready-for-human`**: The issue is fully specified but requires human implementation.
*   **`wontfix`**: The issue will not be actioned.

These labels directly correspond to the canonical roles within the triage state machine.
